    • Desenvolvimento Web

    Desenvolvimento web é o termo utilizado para descrever o desenvolvimento de sites, na Internet ou numa intranet.
    O profissional que trabalha desenvolvendo websites pode ser um web designer ou um web developer, o Progamador atua nessa área, com o desevolvimento de códigos e scripts tanto para o funcionamento do servidor, no caso o Back-end, como na interação do usuario com o site que no caso é o Front-end que veremos.
    iremos abordar as áreas de atuação de um desenvolvedor web, as linguagens de progamação mais utilizadas e as principais ferramentas utilizadas pelos desenvolvedores Back-end e front-end e full-stack, que iremos explicar.

    Front-End

    Desenvolvimento web front-end é o desenvolvimento da interface gráfica do usuário de um site, através do uso de HTML, CSS e JavaScript, para que os usuários possam visualizar e interagir com esse site. O Front-End é área que está ligada com a interação do usuario com a aplicação, site, ou aplicativo é a parte visual, no qual o usuario do aplicativo pode ter acesso as diversas funcionalidades e funções do sistema.
    Exemplos: Um botão, faz parte da interação do usuario com o sistema, o designer,paginas.

    JavaScript

    JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web. O JavaScripit é um lingugagem de progamação, não de marcação e estilo como HTML e css, oque diferencia é que uma linguagem de progamção utiliza códigos para o desenvolvimento de funções e o computador traduz com linguagem binaria que são os bits.(010100101).

    Back-end

    Em uma abordagem mais técnica, o back-end envolve servidor, banco de dados e aplicação. Desta forma, o desenvolvedor na área é responsável por construir e manter esses componentes juntos para que as informações sejam organizadas e armazenadas corretamente, o que permite que uma página funcione bem, de maneira segura e se mantenha no ar para os usuários acessarem. Por exemplo, quando você acessa um site na web, o servidor dessa determinada página envia todas as informações necessárias para que ela se torne visível e você consiga acessá-la.     Python

    Python é uma linguagem de programação de alto nível, interpretada de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi lançada por Guido van Rossum em 1991, python por ser muitos fácil de se usar, é uma das escolhas para pessoas que desejam começar e entrar no mundo da progamação, muito utulizado para o desenvolvimento back-end.

    PHP

    PHP é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web. PHP é uma linguagem de script do tipo server-side com diversos propósitos. Porém, ela é principalmente utilizada para gerar conteúdos dinâmicos num site. Trata-se de uma linguagem altamente popular devido à sua natureza de código aberto e suas funcionalidades versáteis.

    Java

    Java é uma linguagem de programação orientada a objetos desenvolvida na década de 90 por uma equipe de programadores chefiada por James Gosling, na empresa Sun Microsystems, que em 2008 foi adquirido pela empresa Oracle Corporation. Java é uma linguagem multiplataforma, orientada a objetos e centrada em rede que pode ser usada como uma plataforma em si. É uma linguagem de programação rápida, segura e confiável para codificar tudo, desde aplicações móveis e software empresarial até aplicações de big data e tecnologias do servidor.
